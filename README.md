# 📦 TP0 — Inventory Management App
### JSP · Servlet · Hibernate · Jakarta EE · DAO Générique
 
---
 
## 🎯 Objectif
 
Application web complète de gestion d'inventaire développée avec les technologies Java EE modernes. Ce projet démontre l'intégration de JSP, Servlet et Hibernate avec un pattern DAO générique et deux entités métier.
 
---
 
## 🛠️ Technologies utilisées
 
| Technologie | Version | Rôle |
|---|---|---|
| ☕ Java | JDK 11+ | Langage principal |
| 🌐 Jakarta EE | 9.1.0 | API Web (Servlet, JSP) |
| 🔥 Hibernate ORM | 6.0.0.Final | ORM / Persistance |
| 🐬 MySQL (XAMPP) | 8.x | Base de données |
| 🐱 Apache Tomcat | 11.0.20 | Serveur d'application |
| 📦 Maven | 3.x | Gestion des dépendances |
| 🎨 Google Fonts | Playfair Display / Cormorant | Typographie |
 
---

## ⚙️ Prérequis
 
- ✅ JDK 11 ou supérieur
- ✅ Apache Tomcat 10+ (compatible Jakarta EE)
- ✅ XAMPP avec MySQL démarré
- ✅ Maven installé
- ✅ IntelliJ IDEA + plugin SmartTomcat
